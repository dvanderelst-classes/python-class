    \paragraph{Wilcoxon signed-rank test}\label{wilcoxon-signed-rank-test}

The Wilcoxon signed-rank test tests the null hypothesis that two related
paired samples come from the same distribution. In particular, it tests
whether the distribution of the differences x - y is symmetric about
zero. It is a non-parametric version of the paired T-test.

http://users.sussex.ac.uk/\textasciitilde{}grahamh/RM1web/WilcoxonHandoout2011.pdf
